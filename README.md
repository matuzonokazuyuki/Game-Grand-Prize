このゲームの名前はバルーンミッション

このゲームはまだ足りない部分がありますので今後のブラッシュアップです。

このゲームは3D横軸のアクションパズルゲームです。
プレイヤーはキャラクターを操作しギミックを解除と罠を避け、ゴールまで進むゲームです。

このゲームはコントローラー推奨ですか、キーボードとマウスでもできます。

操作方法：

コントローラー：
・左右移動
左スティック
・上移動
△
・下移動
X
・アイテムを持つ/放つ
O
・スキル
□

・左右移動
A/D
・上移動

・下移動
